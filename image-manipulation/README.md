# IMAGE MANIPULATION #

A python script to scale your images.

## Requirements ##

```
$ pip3 install -r requirements.txt
```

## Usage ##

```
python3 main.py
```

Choose your mode to begin:

```
1. Resize Single Image
2. Resize all the Images in a directory
```
